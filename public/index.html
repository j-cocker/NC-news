Topics

/api/topics

GET

Serves an array of all topics:

Example Output

{"topics": [{ "slug": "football", "description": "Footie!" }]}

Articles

/api/articles

GET

Serves an array of all articles, with the comment count, sorted from most to least recent.

Example Output

{ "articles": [ { "title": "Seafood substitutions are increasing", "topic": "cooking", "author": "weegembump", "created_at": "2018-05-30T15:59:13.341Z", "votes": 0, "comment_count": 6 } ] }
